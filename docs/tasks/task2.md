# Задание 2. Технические задания для художника

> Внимание! Данное задание выполняется на основе **вашего** игрового проекта.

---

## Цель задания

Составить полный пакет технических заданий (ТЗ) для создания визуальных элементов игры: персонажей, локаций, интерфейса и ассетов, включая промпты для нейросетевой генерации.

**Техническое задание (ТЗ)** — документ, по которому художник (или вы сами) создаёт визуальный элемент. Хорошее ТЗ позволяет получить именно то, что нужно, и избежать многократных переделок.

> **Правило для работы:** в основном тексте размещается **по одному** демонстрационному ТЗ каждого вида. Остальные материалы — в Приложение.

## 2.3. ТЗ на интерфейс: игровой HUD  
**HUD (Heads-Up Display)** — постоянно видимая информация на экране во время игры: здоровье, очки, инвентарь, мини-карта.

| Параметр | Ваше описание |
|----------|--------------|
| Название экрана / элемента | Игровой HUD |
| Какую информацию отображает | Здоровье Артура (сердца), шкала выносливости (stamina), счётчик осколков Экскалибура, активная способность (иконка), мини-карта (угол) |
| Расположение каждого элемента | Верх-лево: 5 сердец (здоровье). Верх-право: счётчик осколков [X/5] + мини-карта. Низ-центр: шкала выносливости (горизонтальная полоса). Низ-право: иконка активной способности |
| Шрифт: название / гарнитура, кегль, начертание | "MedievalSharp" (Google Fonts), 18 pt для значений счётчиков, 14 pt для подписей, Bold |
| Цветовые акценты (ссылка на палитру) | Рамки и разделители: `#D4AF37` (золото). Текст: `#F5F0E8` (тёплый белый). Сердца: `#CC2200` (полные) / `#4A1A1A` (пустые) |
| Иконки: стиль, размер в пикселях | Плоские с золотым контуром, 32×32 px; осколки — 24×24 px |
| Полоска здоровья: тип | ☐ цифры ☐ полоска ☑ сердца ☐ другое |
| Состояния элементов (нормальное / предупреждение / критическое) | Нормальное: белые сердца. Предупреждение (≤ 2 сердец): медленная пульсация красным. Критическое (1 сердце): частая пульсация + красная виньетка по краям экрана |
| Анимации HUD | Получение урона: красная вспышка на 0.2 сек. Сбор осколка: иконка счётчика крутится + звонок. Использование выносливости: полоса мигает оранжевым при опустошении |
| Адаптация под разрешения | Основное: 1920×1080 (16:9). Дополнительно: 1280×720 — элементы масштабируются пропорционально через Canvas Scaler |
| Canvas mode в Unity | ☑ Screen Space Overlay ☐ Screen Space Camera ☐ World Space |
| Шрифт в Unity | ☑ TextMeshPro ☐ Legacy Text |

**Схема расположения элементов HUD:**

```
┌─────────────────────────────────────────────────────┐
│ ♥ ♥ ♥ ♥ ♥          [мини-карта]   [осколки: 1/5]  │
│                                                     │
│                                                     │
│                  (игровая сцена)                    │
│                                                     │
│            [════════════════] stamina               │
│                                          [⚡ ability]│
└─────────────────────────────────────────────────────┘
```
