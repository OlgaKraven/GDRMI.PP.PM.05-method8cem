# Задание 1. Художественный стиль и визуальная концепция

> Внимание! Ниже приведён **заполненный пример** для игры «Приключение короля Артура». Студент выполняет задание применительно к своему проекту по аналогичной структуре.

---

## Цель задания

Разработать и задокументировать полную визуальную концепцию игрового продукта — от художественного стиля до цветовой палитры и мудборда.

---

## 1.1. Описание художественного стиля

### Аспект 1. Общая эстетика и настроение

Игра «Приключение короля Артура» создаёт атмосферу тёмного эпического фэнтези с нотками меланхолии и надежды. С первых секунд игрок ощущает тяжесть потери — Камелот разрушен, земли прокляты, а легендарный меч утерян. Преобладают холодные сине-фиолетовые оттенки, разбавленные редкими вспышками золотого света — символами того, что было, и того, что ещё можно спасти. Игровое окружение давит, но не лишает воли: каждый найденный осколок Экскалибура — это проблеск надежды. Общее настроение — «одинокий герой против тьмы», схожее с первыми часами *Hollow Knight* или атмосферой *Dark Souls*, но без летальной жёсткости последнего.

### Аспект 2. Тип и характер графики

Используется **2D-графика в стиле dark fantasy cel-shading**. Персонажи имеют чёткие, слегка стилизованные контуры, напоминающие иллюстрации к средневековым манускриптам. Фоны — многослойные параллакс-сцены с детализированным окружением: поваленные колонны, мох на камне, мерцающие руны. Тени мягкие, с эффектом «освещения свечой». Спецэффекты (магия, удары, частицы) яркие и читаемые — контрастируют с приглушённым фоном, сразу притягивая взгляд. Анимации персонажей плавные, с чётко читаемыми ключевыми позами (idle, attack, hurt).

### Аспект 3. Связь стиля с жанром и сеттингом

Тёмный cel-shaded стиль органично работает на жанр action-RPG и сеттинг артуровских легенд по нескольким причинам. Во-первых, он обеспечивает **читаемость** в динамичном бою: контурные персонажи чётко выделяются на фоне даже при высокой скорости. Во-вторых, тёмная цветовая гамма усиливает ощущение угрозы и «падения» мира — что соответствует нарративу о проклятии. В-третьих, стилизация под рисованные иллюстрации отсылает к средневековым манускриптам, в которых и зафиксированы артуровские легенды, создавая культурный резонанс у образованной аудитории. Наконец, 2D-графика технически реализуема в заданные сроки: команда не тратит время на 3D-моделирование и риги.

### Аспект 4. Источники вдохновения

| Игра-вдохновение | Что берём в качестве референса | Как адаптируем под свой проект |
|:----------------:|-------------------------------|-------------------------------|
| **Hollow Knight** | Силуэтный стиль главного персонажа; читаемость на тёмном фоне; атмосфера угасшего мира | Упрощаю силуэт Артура: широкие плечи, корона, плащ. Добавляю золотые акценты — остаток величия Камелота |
| **Ori and the Blind Forest** | Многослойный параллакс фонов; точечные источники света как центры внимания | Применяю Point Light 2D от факелов и рун; фоны делаю многослойными (3–4 слоя) для глубины |
| **Dead Cells** | Чёткая читаемость врагов; анимации с акцентированными ключевыми позами | Противники Морганы имеют фиолетовое свечение — мгновенно считываются как «магические / враждебные» |
| **Gris** | Эмоциональная цветовая прогрессия: первый акт — холодный, финал — тёплый | В «Приключении Артура»: начало — синий/серый Камелот, финал — золотой свет возрождённого Экскалибура |

### Аспект 5. Техническая реализация в Unity

| Инструмент / возможность Unity | Как используется в проекте |
|-------------------------------|---------------------------|
| **URP (Universal Render Pipeline)** | Основной рендер; поддержка 2D-освещения и постобработки |
| **Global Light 2D** | Общий тусклый синий свет — базовая освещённость мира |
| **Point Light 2D** | Факелы, руны, Экскалибур, магические эффекты — локальные источники света |
| **Post Processing Stack (URP Volume)** | Bloom — свечение магии и золота; Vignette — затемнение краёв экрана в бою |
| **Sprite Atlas** | Объединение спрайтов персонажей и врагов в атласы для снижения Draw Calls |
| **Particle System** | Эффекты ударов, магии Морганы, частицы при сборе рун |
| **Animator Controller** | FSM для каждого персонажа: idle → walk → run → attack → hurt → death |
| **2D Shader Graph (Custom Lit Sprite)** | Cel-shading shader с настраиваемыми уровнями тени (2 шага) |

| Функция цвета | HEX-код | Описание / когда используется |
|:-------------:|:-------:|------------------------------|
| Основной фон (ночное небо, камень) | `#1A1A2E` | Фоны локаций, небо, затемнённые зоны; создаёт ощущение ночи |
| Основной цвет персонажа (Артур) | `#C0A060` | Золочёная броня, детали плаща; символ королевского достоинства |
| Вторичный цвет персонажа (плащ) | `#4A3F6B` | Тёмно-синий плащ; выделяет Артура на тёмном фоне |
| Цвет врагов / опасности (магия Морганы) | `#8B2FC9` | Фиолетовое свечение у всех проклятых существ; мгновенно читается как угроза |
| Цвет безопасных / интерактивных объектов | `#50C878` | Зелёные руны, точки сохранения, предметы; «здесь можно взаимодействовать» |
| Основной цвет UI / HUD | `#D4AF37` | Рамки, иконки, разделители — золото Камелота |
| Цвет текста интерфейса | `#F5F0E8` | Все подписи и тексты — тёплый белый, читается на тёмном фоне |
| Цвет подсветки при выборе | `#FFD700` | Кнопки при наведении, выделение предметов — яркое золото |
| Экскалибур / магия света | `#E8F4FF` | Эффекты удара Экскалибура, разблокировка способностей — холодный белый свет |

> **Проверка контрастности:** `#F5F0E8` на `#D4AF37` = 4.6:1 ✓ (WCAG AA). `#F5F0E8` на `#1A1A2E` = 11.2:1 ✓ (WCAG AAA).

### Итоговое описание визуальной концепции

«Приключение короля Артура» визуально строится на контрасте **тьмы и угасающего золота**. Мир игры — это Британия после катастрофы: холодные сине-фиолетовые тона, разрушенные колонны и мерцающие проклятые руны создают ощущение мира, потерявшего своё величие. На этом фоне главный герой — Артур в золочёной броне — выглядит последним символом прежней эпохи.

Визуальный язык построен на строгой цветовой семантике: фиолетовый — всегда угроза (враги и магия Морганы), зелёный — безопасность и взаимодействие (руны и точки сохранения), золотой — прогресс и надежда (предметы, UI, Экскалибур). Это позволяет игроку читать экран интуитивно без туториальных подсказок.

Стилистически игра вдохновлена *Hollow Knight* (читаемость в тёмной среде) и *Ori and the Blind Forest* (эмоциональная прогрессия цвета). Техническая реализация в Unity URP включает 2D Point Light для создания атмосферных источников света, cel-shading шейдер с двухступенчатым затенением и Bloom для магических эффектов.
